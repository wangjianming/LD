Слияние слоев

Этот инструмент копирует объекты из двух слоев в новый слой.Объединяемые слои должны содержать объекты одного типа (полигоны, линии или точки).Способом объединения и копирования полей можно управлять.Например:

  • у вас имеется три слоя: Англия, Уэльс и Шотландия,- а вы хотите создать один слой Великобритании.
  • У вас есть 12 слоев, каждый из которых содержит информацию о земельных участках в смежных тауншипах (участок площадью 36 кв. миль). Я хочу объединить их в один слой, сохранив только поля с одинаковыми названиями и типами из всех 12 слоев.

Если опция Использовать текущий экстент карты (Use current map extent) включена, объединяются только те объекты, которые отображаются в текущем экстенте. Если опция отключена, будут объединены все объекты обоих слоев, даже если они находятся вне текущего экстента карты.

Выберите слой

Выберите слой для объединения со слоем анализа.

Обновить объединенные поля

По умолчанию, все поля обоих слоев переносятся в выходные данные.Можно определить, что будет происходить с полями в результирующем слое:

  • Переименовать – Поле будет переименовано в выходном слое.
  • Удалить – Поле будет удалено из результата.
  • Сопоставить – Имена полей приводятся в соответствие друг другу. Значения поля из слоя для объединения будут скопированы в соответствующие выходные поля.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Имя слоя по умолчанию зависит от типа анализа и имени слоя анализа.Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
