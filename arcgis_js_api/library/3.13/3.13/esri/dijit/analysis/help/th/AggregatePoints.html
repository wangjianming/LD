รวมจุด

เครื่องมือนี้ทำงานกับชั้นข้อมูลประเภทจุด (ชั้นข้อมูลที่ใช้วิเคราะห์) และชั้นข้อมูลเชิงพื้นที่โดยจะพิจารณาจากข้อมูลเชิงจุดที่อยู่ภายในแต่ละพื้นที่หลังจากที่ตรวจสอบความสัมพันธ์เชิงพื้นที่ของจุดที่อยู่ในพื้นที่, ค่าทางสถิติของจุดที่อยู่ในพื้นที่จะถูกคำนวณและเก็บไว้ในพื้นที่นั้นๆค่าทางสถิติพื้นที่ฐานเป็นการนับจำนวนของจุดที่อยู่ภายในพื้นที่ แต่คุณสามารถใช้ค่าสถิติอื่นได้เช่นกัน

ตัวอย่างเช่น สมมติว่าคุณมีข้อมูลตำแหน่งร้านกาแฟและมีข้อมูลพื้นที่เขต และคุณต้องการสรุปจำนวนยอดขายในเขตนั้นโดยตำแหน่งร้านกาแฟมี TOTAL_SALES ข้อมูลเชิงบรรยายที่คุณต้องการสรุปจำนวนทั้งหมด TOTAL_SALES ภายในขอบเขตของแต่ละเมือง หรือยอดขายต่ำสุด สูงสุด TOTAL_SALES ในแต่ละขอบเขตของเมือง หรือค่าเบี่ยงเบนมาตรฐานของยอดขายในแต่ละเมืองได้

ผลลัพธ์ คือ แผนที่ที่แสดงสีของแต่ละเขต ซึ่งขึ้นอยู่กับจำนวนของตำแหน่งร้านกาแฟต่อขนาดของพื้นที่การแบ่งการคำนวณโดยใช้พื้นที่เป็นมาตรฐานที่รู้จักกันทั่วไป และยังเป็นเทคนิคในการทำแผนที่ซึ่งใช้ในการแสดงผลข้อมูลเชิงพื้นที่การทำให้เป็นมาตรฐานช่วยให้คุณเข้าใจแบบรูปของพื้นที่มากขึ้น เนื่องจากขนาดของพื้นที่ถูกนำไปใช้ในการอธิบายเมื่อมีการเปรียบเทียบพื้นที่แต่ละแห่ง

ถ้า ใช้ขอบเขตแผนที่ปัจจุบัน ถูกเลือกไว้ จะมีการใช้ข้อมูลเพื่อการวิเคราะห์นี้เฉพาะข้อมูลที่อยู่ในพื้นที่จากขอบเขตแผนที่ที่เห็นอยู่ในปัจจุบันมาทำการวิเคราะห์เท่านั้น ถ้าไม่เลือก ฟีเจอร์ทั้งหมดที่อยู่ในชั้นข้อมูลวิเคราะห์และชั้นข้อมูลพื้นที่จะถูกนำไปวิเคราะห์ แม้ว่าข้อมูลนั้นจะอยู่นอกขอบเขตแผนที่ที่แสดงอยู่ในปัจจุบันก็ตาม

เลือกพื้นที่

เลือกชั้นข้อมูลพื้นที่จากเว็บแผนที่ เพื่อรวบรวมข้อมูลจากจุด ถ้าไม่มีขอบเขตของชั้นข้อมูลในเว็บแมพ คุณสามารถเลือกจากรายการขอบเขตของชั้นข้อมูลที่มีให้ใน ArcGIS Onlineจุดใดๆ ที่ตกอยูภาย่ในขอบเขตของพื้นที่ในชั้นข้อมูลนี้จะถูกนัด และสรุปผลรวมทางสถิติประเภทต่างๆ ตามที่เลือกไว้

รักษาพื้นที่ไว้โดยไม่มีจุด

กำหนดให้ พื้นที่ที่ไม่มีข้อมูลจุด แสดงในชั้นข้อมูลผลลัพธ์ถ้าไม่เลือก, พื้นที่ที่ไม่มีข้อมูลจุดจะไม่ปรากฎในผลลัพธ์ถ้าไม่เลือก, พื้นที่ที่ไม่มีข้อมูลจุดจะปรากฎในผลลัพธ์

เพิ่มข้อมูลสถิติ

คุณสามารถคำนวณค่าทางสถิติในตารางคุณลักษณะของชั้นข้อมูลจุด เช่น ค่ารวม, ค่าเฉลี่ย, ค่าน้อยที่สุด, ค่ามากที่สุด และค่าความเบี่ยงเบนมาตรฐานชั้นข้อมูลผลลัพธ์จะประกอบด้วยค่าข้อมูลคุณลักษณะใหม่หลังจากการคำนวณทางสถิติค่าทางสถิติใดๆ จะสามารถเพิ่มขึ้นโดยเลือกจากข้อมูลคุณลักษณะและค่าสถิติ (กล่องรายการตัวเลือกแบบดึงลงใหม่จะปรากฎขั้น)

เลือกฟีลด์ที่ต้องการให้รวมกลุ่มกัน

คุณสามารถสร้างกลุ่มทางสถิติโดยใช้ตารางคุณลักษณะในชั้นข้อมูลสถิติตัวอย่างเช่น เมื่อคุณต้องการรวบรวมข้อมูลการเกิดเหตุอาชญากรรมในพื้นที่ใกล้เคียง จากข้อมูลเชิงบรรยายที่มี Crime_type จากทั้งอาชญากรรมที่แตกต่างกันห้าประเภท ประเภทอาชญากรรมแต่ละประเภทจะถูกใช้ในการจัดกลุ่ม เพื่อทำการคำนวนค่าทางสถิติให้จากที่เลือกไว้แต่ละค่า Crime_type. ค่าสถิติของแต่ละกลุ่ม เช่นเดียวกับจำนวนนับของข้อมูลจุด จะนำไปแสดงใน pop-up ของชั้นข้อมูลผลลัพธ์เมื่อคุณเลือกกลุ่มข้อมูลเชิงบรรยาย จะได้ผลลัพธ์ออกมาสองอย่าง คือ ชั้นข้อมูลผลลัพธ์ และตารางผลลัพธ์ ซึ่งมีค่าทางสถิติที่ได้แยกกันในแต่ละกลุ่ม

เพิ่มค่าจำนวนส่วนน้อย จำนวนส่วนมาก

ตัวเลือกแบบกาเครื่องหมายนี้จะเปิดใช้งานได้ เมื่อคุณเลือกฟิลด์สำหรับกำหนดกลุ่ม เมื่อคุณต้องการค้นหาด้วยค่าในตารางของแต่ละกลุ่มที่เป็นค่าส่วนน้อย (มีจำนวนค่านั้นน้อยที่สุด) หรือค่าส่วนมาก (มีจำนวนค่านั้นมากที่สุด) ในแต่ละขอบเขตพื้นที่ ให้เลือก เพิ่มค่าจำนวนส่วนน้อย จำนวนส่วนมาก. เมื่อคุณเลือก เพิ่มค่าจำนวนส่วนน้อย จำนวนส่วนมาก ฟิลด์ใหม่สองฟิลด์จะถูกเพิ่มเข้าไปในชั้นข้อมูลผลลัพธ์ของคุณ เมื่อคุณเลือก เพิ่มเปอร์เซนต์ จะมีฟิลด์เพิ่มอีกสองฟิลด์ในชั้นข้อมูลผลลัพธ์ของคุณ โดยเป็นฟิลด์สำหรับค่าเปอร์เซ็นต์ของส่วนน้อย และค่าเปอร์เซ็นต์ของส่วนมาก ในแต่ละกลุ่ม

ตัวอย่างเช่น ถ้าคุณเป็นนักวางผังเมืองและคุณต้องการค้นหาประเภทของต้นไม้ที่มีอยู่เป็นส่วนน้อย และที่มีอยู่เป็นส่วนมากในสวนสาธารณะทั้งหมดของเมือง ข้อมูลสถิติของต้นไม้จะถูกรวมเข้ากับพื้นที่สวนสาธารณะของคุณตามที่ได้กำหนดตัวเลือกไว้

เพิ่มเปอร์เซ็นต์

คุณจะใช้กล่องกาเครื่องหมายนี้ได้หากคุณเลือกฟิลด์ที่จะจัดกลุ่ม เลือก เพิ่มเปอร์เซนต์ หากคุณต้องการค้นหาเปอร์เซ็นต์ของค่าตัวแปรแต่ละค่าภายในแต่ละกลุ่ม ฟิลด์ใหม่จะถูกเพิ่มลงในตารางผลลัพธ์ที่ประกอบด้วยเปอร์เซ็นต์ของค่าตัวแปรแต่ละค่าภายในแต่ละกลุ่ม หาก เพิ่มส่วนน้อย ส่วนใหญ่ ก็จะถูกเลือกด้วย ฟิลด์อีกสองฟิลด์จะถูกเพิ่มลงในชั้นข้อมูลผลลัพธ์ที่ประกอบด้วยเปอร์เซ็นต์ของค่าตัวแปรส่วนน้อยและส่วนใหญ่ภายในแต่ละกลุ่ม

ชื่อชั้นข้อมูลผลลัพธ์

เป็นชื่อชั้นข้อมูลที่จะถูกสร้างใน เนื้อหาของฉัน และเพิ่มเข้าไปในแผนที่ ชื่อตั้งต้นจะอ้างอิงจากประเภทการวิเคราะห์และชื่อชั้นข้อมูลที่วิเคราะห์ ถ้าชั้นข้อมูลนี้มีอยู่แล้ว คุณจะถูกถามให้ตั้งชื่อใหม่

ช้ บันทึกผลลัพธ์ใน รายการตัวเลือก เพื่อระบุชื่อของโฟล์เดอร์ใน เนื้อหาของฉัน เมื่อบันทึกผลลัพธ์
