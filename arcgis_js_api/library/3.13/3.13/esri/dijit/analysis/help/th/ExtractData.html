สร้างข้อมูล

เครื่องมือนี้ คุณสามารถเลือกและดาวน์โหลดข้อมูลจากพื้นที่ที่สนใจชั้นข้อมูลที่คุณเลือกนี้สามารถเพิ่มเป็นไฟล์ Zip หรือไฟล์ csv ได้

ชั้นข้อมูลที่ต้องการแตกข้อมูล

มีชั้นข้อมูลในหน้าเนื้อหาของคุณที่คุณสามารถแยกฟีทเจอร์ได้ชั้นข้อมูลที่ถูกตรวจสอบนั้นถูกแบ่งข้อมูลและรวมผลลัพธ์อยู่ในไฟล์ประเภท zip หรือ csv

เพื่อให้ชั้นข้อมูลที่ปรากฏในรายการที่แตกข้อมูล คุณต้องเป็นเจ้าของข้อมูลหรือได้รับการอนุญาตจากเจ้าของให้สามารถนำข้อมูลออกได้เพื่อเปิดให้ส่งออกข้อมูลของคุณได้ :

  • เปิดระละเอียดไอเท็มของข้อมูล
  • กดปุ่ม แก้ไข เพื่อแก้ไขรายละเอียดของไอเท็ม
  • เลื่อนลงไปที่ส่วนของ คุณสมบัติ
  • เลือก ส่งออกข้อมูล.

ถ้าชั้นข้อมูลที่คุณวิเคราะห์มีไฟล์แนบ ไฟล์แนบเหล่านั้นจะถูกนำออกมาได้เมื่อข้อมูลผลลัพธ์เป็นฐานข้อมูล geodatabase แบบไฟล์ และ การตัดข้อมูล ไม่ถูกเลือกไว้

พื้นที่ศึกษา

พื้นที่ที่กำหนด โดยฟีทเจอร์จะรวมอยู่ในไฟล์ผลลัพธ์เป็นไฟล์ zip หรือ csvคุณสามารถเลือก ได้ดังนี้

  • ขอบเขตแผนที่ที่เห็นอยู่ในปัจจุบัน ( เหมือนหน้าจอ)
  • ชั้นข้อมูลใดชั้นข้อมูลหนึ่งในตารางรายการชั้นข้อมูล
  • วาดกำหนดพื้นที่ศึกษา
ข้อมูลทั้งหมดที่อยู่ภายในขอบเขตของแผนที่จะถูกนำออกมาก สำหรับข้อมูลที่มีพื้นที่ข้าม หรือซ้อนทับขอบเขตแผนที่ แต่ไม่ได้อยู่ภายในพื้นที่ทั้งหมด มี สองตัวเลือกให้คุณกำหนด ดังนี้
  • เลือกข้อมูล—ข้อมูลมีพื้นที่ตัดข้ามพื้นที่ศึกษาจะถูกนำออกมาทั้งชิ้น
  • ตัดข้อมูล—เฉพาะบางส่วนของข้อมูลที่อยู่ภายในพื้นที่ศึกษาจะถูกตัดออกมา

นามสกุลข้อมูลผลลัพธ์

ฟอร์แม็ตของข้อมูลจะถูกแยกเมื่อนำข้อมูลเส้นหรือพื้นที่ออกไปเป็นไฟล์ csv ข้อมูลคุณลักษณะที่ไม่ใช่เชิงพื้นที่เท่านั้นที่จะถูกแยกเมื่อนำออกข้อมูลแบบจุด ทั้งข้อมูลคุณลักษณะแบบเชิงพื้นที่และไม่เชิงพื้นที่ จะถูกแยก

ชื่อไฟล์ผลลัพธ์

เป็นชื่อชั้นข้อมูลที่จะถูกสร้างใน เนื้อหาของฉัน ชื่อไอเท็ม จะอิงตามการเลือก รูปแบบผลลัพธ์ ถ้าไอเท็มนี้มีอยู่แล้ว คุณจะถูกถามให้ตั้งชื่อใหม่

ใช้ บันทึกผลลัพธ์ใน รายการตัวเลือก เพื่อระบุชื่อของโฟล์เดอร์ใน เนื้อหาของฉัน เมื่อบันทึกผลลัพธ์
